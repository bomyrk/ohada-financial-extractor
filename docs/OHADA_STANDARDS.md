
# OHADA Accounting Standards: Account Reference Guide



This document defines the standard account codes used in OHADA-compliant financial statements across 18 African countries.



## Overview



OHADA (Acte uniforme relatif au droit comptable et à l'information financière) establishes uniform accounting standards for organizations in member countries. This library implements the standardized account coding system.

## Value Types for Assets



Balance sheet assets track three value types:



- **Brut (Gross)**: Original acquisition cost

- **Amortissement (Amortization)**: Accumulated depreciation

- **Net**: Book value = Gross - Amortization



Non-asset statements track only **Net** values.



## Validation Rules



### Balance Sheet Equation



Total Assets (BZ) = Total Liabilities & Equity (DZ)





### Income Statement Link



Net Income (XI) = Retained Earnings in Balance Sheet (CJ)





### Cash Flow Check



Net Change in Cash (ZG) = Operating CF (ZB) + Investing CF (ZC) + Financing CF (ZE)
