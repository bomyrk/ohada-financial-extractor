# Security Policy

## Supported Versions
We actively maintain and support the latest minor version of `ohada-financial-extractor`.  
Older versions may not receive security updates.

| Version | Supported |
|--------|-----------|
| 0.1.x  | Yes       |
| < 0.1  | No        |

## Reporting a Vulnerability
If you discover a security vulnerability, please report it responsibly.

**Do not open a public GitHub issue.**

Instead, contact us directly at:

📧 security@bkresearchandanalytics.com

📧 contact@bkresearchandanalytics.com

Please include:
- a description of the vulnerability  
- steps to reproduce (if applicable)  
- potential impact  
- suggested remediation (optional)

We will acknowledge your report within **72 hours** and keep you informed of the resolution timeline.

## Responsible Disclosure
We kindly ask that you:
- give us reasonable time to investigate and fix the issue  
- avoid publicly disclosing details until a patch is released  

We appreciate your help in keeping the OHADA financial ecosystem secure.
